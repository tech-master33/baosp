# BAOSP Changelog

This file is updated automatically on every nightly build.
Each entry lists the commits that went into that night's release of both the screen reader and TTS engine.

---

## New Apps Added (June 2, 2026)

### baosp-reader-assistant
Enhanced document and email reader with advanced text-to-speech options and reading mode features

### baosp-voice-control
Voice control and voice command app for hands-free device operation

### baosp-notification-manager
Enhanced notification management with auditory and haptic feedback customization

### baosp-accessibility-testing
Testing and diagnostics tool for developers to validate accessibility compliance

### baosp-panel (Updated)
Quick-access control panel with expanded features for speech rate, flashlight, volume and screen reader status management
